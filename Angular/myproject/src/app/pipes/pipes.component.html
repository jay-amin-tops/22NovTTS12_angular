Pipes Details
DatePipe Formats a date value according to locale rules.

{ { value_expression | date [ : format [ : timezone [ : locale ] ] ]} }

Date Without pipe : {{todaydate}}

Date with pipe : {{todaydate|date:'d/M/y'}}

Date with pipe : {{todaydate | date:'shortTime'}}
UpperCasePipe Transforms text to all upper case.

Uppercase Pipe

{{title | uppercase}}
LowerCasePipe Transforms text to all lower case.

Lowercase Pipe

{{title | lowercase}}
CurrencyPipe Transforms a number to a currency string, formatted according to locale rules.

Currency Pipe

{{6589.23 | currency:"USD"}}
{{6589.23 | currency:"USD":true}} //Boolean true is used to get the sign of the currency.
DecimalPipe Transforms a number into a string with a decimal point, formatted according to locale rules.

Decimal Pipe

{{ 454.78787814 | number: '3.4-4' }} // 3 is for main integer, 4 -4 are for integers to be displayed.
PercentPipe Transforms a number to a percentage string, formatted according to locale rules.

Percent Pipe

{{00.54565 | percent}}
